# Deep Learning Project - Université

Projet complet de Deep Learning comportant trois parties :
1. MLP pour données tabulaires (Breast Cancer Wisconsin)
2. CNN pour classification d'images (Fashion-MNIST)
3. RNN/LSTM/GRU/Seq2Seq pour données séquentielles (IMDb)

## Structure du projet

```
deeplearning/
├── part1_mlp/                    # Partie I : MLP
│   ├── theory_mlp.md            # Théorie détaillée
│   ├── mlp_breast_cancer.ipynb  # Notebook principal
│   └── figures/                 # Visualisations
├── part2_cnn/                    # Partie II : CNN
│   ├── theory_cnn.md            # Théorie détaillée
│   ├── cnn_fashion_mnist.ipynb  # Notebook principal
│   └── figures/                 # Visualisations
├── part3_rnn/                    # Partie III : RNN/LSTM/GRU
│   ├── theory_rnn.md            # Théorie détaillée
│   ├── rnn_imdb.ipynb           # Notebook principal
│   └── figures/                 # Visualisations
├── report/                       # Rapport académique
│   ├── introduction.md
│   ├── part1_report.md
│   ├── part2_report.md
│   ├── part3_report.md
│   └── final_discussion.md
└── requirements.txt              # Dépendances
```

## Installation

```bash
pip install -r requirements.txt
```

## Exécution

Lancer les notebooks Jupyter dans l'ordre :
1. `part1_mlp/mlp_breast_cancer.ipynb`
2. `part2_cnn/cnn_fashion_mnist.ipynb`
3. `part3_rnn/rnn_imdb.ipynb`
